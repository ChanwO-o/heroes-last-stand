Heroes: The Last Stand is a tower based defense game which is being built from scratch in Unity. This is a strategy tower defense game like never before! Unlike other traditional tower defense games, Heroes: The Last Stand allows the player to control both heroes and towers, creating a unique gameplay experience that features real-time strategy elements.

Heroes: The Last Stand our goal is to allow players to control a hero which will help defend their castle. Players will be able to control their hero and in turn move that player around the screen to combat the enemy force.
